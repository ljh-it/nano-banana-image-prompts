# Quick Start

## Prompt
[Subject] + [Style Anchor] + [Lighting] + [Camera/Lens] + [Detail] + [Environment] + [Composition] + [Text(optional)]

## Two knobs you should always control
1) Composition: close-up / half-body / wide shot / top-down
2) Lighting: soft diffused / rim light / golden hour / studio

## Recommended defaults
- Add lens hint: 35mm / 50mm / 85mm
- Add realism anchor: natural skin texture, pores, realistic fabric threads
- Add depth: shallow depth of field, subtle bokeh
